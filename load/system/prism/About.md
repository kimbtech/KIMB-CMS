# PrismJS

Dieser Ordner enthält PrismJS für das KIMB-CMS.  
Sie können auf einer Seite im Header `<!-- PrismJS -->` einfügen
und Quellcode wie folgt anzeigen:

```
	<pre><code class="language-css">
		p { color: red }
	</code></pre>
```
